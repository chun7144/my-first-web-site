HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page by using tags and attributes. Tags are used to identify different parts of a web page, such as headings, paragraphs, and images. Attributes are used to provide additional information about a tag, such as the size of an image or the color of a heading.
